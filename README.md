# brewchestrator
An app to manage, analyze, and orchestrate brew days

## Features
- Create and store beer recipes
- Search and add ingredients (hops, grains, yeast)
- Automatic calculation of OG, IBU, and SRM
- BJCP style selection and visualization
- Custom ingredient support
- Style guideline comparison charts

## Setup
```bash
pip install -r requirements.txt
python app.py
```

Visit http://localhost:5000

## Usage
1. Click "Create New Recipe"
2. Enter recipe name, brewer, and batch size
3. Select a BJCP style
4. Search and add grains, hops, and yeast
5. Customize ingredient amounts and properties
6. View calculated stats and style comparison
7. Save your recipe
